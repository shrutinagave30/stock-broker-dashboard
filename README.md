# Stock Broker Dashboard

## Overview
The Stock Broker Dashboard is a web-based application designed to simulate a real-time stock monitoring platform. It enables users to track selected stock tickers, visualize price movements, and observe dynamic updates through an interactive and responsive dashboard.
This project is developed as part of an internship assignment to demonstrate practical skills in modern web development, UI design, and cloud deployment.

## Technologies Used

### Frontend
- **Next.js** – React framework for optimized rendering and routing  
- **React.js** – Component-based UI development  
- **TypeScript** – Static typing for better code reliability  
- **Tailwind CSS** – Utility-first CSS framework for responsive design  

### Data & State Management
- **SWR** – Real-time data fetching and caching  
- **Recharts** – Data visualization and charting  
- **React Hook Form** – Form handling and validation  
- **Zod** – Schema-based input validation  

### Tools & Deployment
- **Git & GitHub** – Version control and source management  
- **Vercel** – Cloud deployment and hosting platform

## Installation & Local Setup
```bash
# Clone the repository
git clone https://github.com/shrutinagave30/stock-broker-dashboard.git

# Navigate to project directory
cd stock-broker-dashboard

# Install dependencies
npm install

# Start development server
npm run dev
Open http://localhost:3000 in your browser.

**Deployment**
The project is deployed using Vercel with continuous deployment enabled.
Any push to the main branch automatically triggers a new production build.
